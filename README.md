# Conjugueur FR — тренажёр французских глаголов

## Описание
Conjugueur FR — это одностраничное веб‑приложение, помогающее изучать спряжение
ключевых французских глаголов. Приложение совмещает интерактивный справочник и
режим тренировки, чтобы ускорить запоминание форм и освоение произношения.

## Возможности
- Подробные таблицы спряжения для пяти высокочастотных глаголов (être, avoir,
  aller, venir, prendre, voir, vouloir) в трёх распространённых временах:
  présent, imparfait и passé composé, включая вспомогательные глаголы и
  причастия прошедшего времени.
- Отображение транскрипции IPA и подсказок русской транскрипцией для каждой
  формы, что помогает закрепить произношение.
- Интерактивный переключатель темы (светлая/тёмная), который делает занятие
  комфортным в любое время суток.
- Режим тренировки на 15 раундов: приложение предлагает глагол, время и
  местоимение, а пользователь вспоминает форму, сверяется с правильным ответом и
  отслеживает прогресс.
- Быстрый переход между справочником и тренировкой, повторение завершённой
  сессии одним кликом.

## Как пользоваться
1. Откройте файл `index.html` в современном браузере (Chrome, Firefox, Safari,
   Edge). Приложение не требует серверной части и работает полностью офлайн.
2. На вкладке «Справочник» выберите глагол и время — таблица автоматически
   покажет нужные формы.
3. Нажмите «Начать тренировку», чтобы перейти в практический режим. Введите
   ответ в поле и нажмите «Показать ответ», чтобы свериться с правильной формой.
4. После 15 раундов можно мгновенно повторить тренировку или вернуться к
   справочнику.

## Структура проекта
- `index.html` — единственный файл приложения, содержит разметку, стили и
  JavaScript‑логику: переключение экранов, заполнение таблицы, генерация раундов
  тренировки и хранение словаря глаголов.
- `README.md` — текущий файл с описанием проекта.

## Технологии
- Чистый HTML, CSS и JavaScript без внешних зависимостей.
- Адаптивный дизайн с использованием современного CSS (flexbox, grid,
  CSS custom properties) и поддержкой тёмной темы.
